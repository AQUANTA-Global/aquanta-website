# AQUANTA Web Framework v1.0

Static GitHub Pages framework for AQUANTA Co., Ltd.

## Core files

- `index.html`
- `components/header.html`
- `components/footer.html`
- `assets/css/aquanta-theme.css`
- `assets/js/aquanta-core.js`
- `data/content.json`
- `robots.txt`
- `sitemap.xml`

## Deployment

Upload the contents of this folder to the GitHub Pages repository root.
The `CNAME` file must contain:

```text
aquanta.co.jp
```
